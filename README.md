# TableList

<a href="https://ci.appveyor.com/project/bantikyan/tablelist"><img src="https://ci.appveyor.com/api/projects/status/ectnla0f5r58yg2m/branch/master?svg=true" alt="Build status"></a>

This package gives ASP.Net developers easy way of editing list of items with model binding. This means you can edit elements of your list inline, and all changes will be submited with form.
